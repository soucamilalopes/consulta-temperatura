
# Consulta de Dados Meteorológicos

Programa desenvolvido em Python para leitura, processamento e análise de dados meteorológicos históricos, permitindo consultas e geração de visualizações a partir de registros entre os anos de 1961 e 2016.

🌡️ Consulta de dados de temperatura, precipitação, umidade e vento por intervalo de tempo informado pelo usuário, entre 1961 e 2016;

🌧️ Identificação do mês mais chuvoso de todo o conjunto de dados;

❄️ Médias de temperatura mínima do mês informado pelo usuário em cada um dos anos entre 2006 e 2016;

📊 Geração de gráfico com as médias de temperatura mínima do mês informado pelo usuário em cada um dos anos entre 2006 e 2016;

📈 Média geral da temperatura mínima do mês informado pelo usuário, considerando todos os anos entre 2006 e 2016.

## Como funciona

O programa realiza a leitura e o tratamento de um arquivo contendo dados meteorológicos referentes ao período de 1961 a 2016. A partir das entradas fornecidas pelo usuário, os registros são filtrados e apresentados em modo texto e gráfico, permitindo diferentes tipos de consulta e análise.

## Opções de consulta

- Todos os dados;
- Apenas os dados de precipitação;
- Apenas os dados de temperatura;
- Apenas os dados de umidade e velocidade do vento;
- Consulta das temperaturas mínimas entre os anos de 2006 e 2016.

## Validações implementadas

- Verificação de mês válido para consultas (1 a 12);
- Verificação de ano válido para consultas (1961 a 2016);
- Reentrada de dados em caso de mês inválido;
- Reentrada de dados em caso de ano fora do intervalo permitido;
